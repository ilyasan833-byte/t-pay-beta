# t-pay-beta
An app that simulates a fake payment terminal operating system, with settings that allow you to turn off/on the sound, change the lock screen wallpaper, and access the terminal's menu to enter a fake payment account and method.
приложение которое имитирует ложную операционную систему платёдного терминала, в нёём есть настройки, в котором можно отключить/включить звук, поменять обои экрана блокировки, и само меню терминала в котором можно ввести счёт и способ ложной оплаты.
